Bug 1: Некорректная работа функции capitilize при вводе чисел с пробелом в файле string_utils.py
Предварительные шаги:
Должен быть установлен Pytest
Шаги для выполнения:
1. Открыть файл test_string_utils.py
2. Ввести данные в @pytest.mark.parametrize, содержащие цифры, пробел и слово с маленькой буквы, например "223 blue 223"
3. Ввести в консоли "pytest"
Ожидаемый результат: "223 blue 223" преобразуется в "223 Blue 223", т.е. 1-я буква в слове преобразуется из строчной в заглавную
Фактический результат: 1-я буква в слове не преобразуется из строчной в заглавную
Серьёзность: Critical
Приоритет: High
Окружение: Windows 11 24H2 x64 10.0.26100, Visual Studio Code 1.96.0, Python 3.13.0, Pytest 8.3.4
